# GIF Generation

Generate animated GIFs from HTML with CSS animations or by capturing live web pages.

## Generate from HTML

Create GIFs from HTML with CSS animations.

<CodeGroup>
```bash cURL
curl -X POST https://api.pictify.io/v1/gif \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "html": "<div class=\"container\"><style>@keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } .container { width: 400px; height: 400px; display: flex; align-items: center; justify-content: center; background: #1a1a2e; } .circle { width: 100px; height: 100px; background: #667eea; border-radius: 50%; animation: pulse 1s infinite; }</style><div class=\"circle\"></div></div>",
    "width": 400,
    "height": 400
  }'
```

```typescript Node.js
import Pictify from '@pictify/sdk';

const pictify = new Pictify('pk_live_...');

const result = await pictify.gifs.create({
  html: `
    <div class="container">
      <style>
        @keyframes pulse {
          0% { transform: scale(1); }
          50% { transform: scale(1.1); }
          100% { transform: scale(1); }
        }
        .container {
          width: 400px;
          height: 400px;
          display: flex;
          align-items: center;
          justify-content: center;
          background: #1a1a2e;
        }
        .circle {
          width: 100px;
          height: 100px;
          background: #667eea;
          border-radius: 50%;
          animation: pulse 1s infinite;
        }
      </style>
      <div class="circle"></div>
    </div>
  `,
  width: 400,
  height: 400
});

console.log(result.url);
```

```python Python
from pictify import Pictify

pictify = Pictify('pk_live_...')

result = pictify.gifs.create(
```

## Capture from URL

Record animations from a live web page.

<CodeGroup>
```bash cURL
curl -X POST https://api.pictify.io/v1/gif/capture \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/animated-page",
    "width": 800,
    "height": 600,
    "quality": "high",
    "frameDurationSeconds": 5
  }'
```

```typescript Node.js
const result = await pictify.gifs.capture({
  url: 'https://example.com/animated-page',
  width: 800,
  height: 600,
  quality: 'high',
  frameDurationSeconds: 5
});
```

```python Python
result = pictify.gifs.capture(
    url='https://example.com/animated-page',
    width=800,
    height=600,
    quality='high',
    frame_duration_seconds=5
)
```
</CodeGroup>

## Request Parameters

### HTML Generation

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `html` | string | Yes | HTML with CSS animations |
| `width` | integer | Yes | Output width (1-1200) |
| `height` | integer | Yes | Output height (1-1200) |
| `quality` | string | No | Quality preset (default: `medium`) |
| `frameDurationSeconds` | number | No | Animation duration to capture |

### URL Capture

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `url` | string | Yes | URL to capture |
| `width` | integer | Yes | Output width (1-1200) |
| `height` | integer | Yes | Output height (1-1200) |
| `quality` | string | No | Quality preset (default: `medium`) |
| `frameDurationSeconds` | number | Yes | Recording duration in seconds |
| `selector` | string | No | CSS selector to capture specific element |
| `waitForSelector` | string | No | Wait for element before recording |

## Quality Presets

| Preset | Frame Rate | Max Duration | Best For |
|--------|-----------|--------------|----------|
| `low` | 10 fps | 10 seconds | Small file size, simple animations |
| `medium` | 15 fps | 15 seconds | Balanced quality and size |
| `high` | 24 fps | 30 seconds | Smooth animations, detailed content |

<Note>
Higher quality settings produce larger files. Choose based on your use case and bandwidth requirements.
</Note>

## Examples

### Loading Spinner

```typescript
const result = await pictify.gifs.create({
  html: `
    <div style="width: 200px; height: 200px; display: flex; align-items: center; justify-content: center; background: white;">
      <style>
        @keyframes spin {
          from { transform: rotate(0deg); }
          to { transform: rotate(360deg); }
        }
        .spinner {
          width: 50px;
          height: 50px;
          border: 4px solid #e5e7eb;
          border-top-color: #667eea;
          border-radius: 50%;
          animation: spin 1s linear infinite;
        }
      </style>
      <div class="spinner"></div>
    </div>
  `,
  width: 200,
  height: 200,
  quality: 'medium'
});
```

### Typing Animation

```typescript
const result = await pictify.gifs.create({
  html: `
    <div style="width: 600px; height: 100px; display: flex; align-items: center; padding: 20px; background: #1a1a2e; font-family: monospace;">
      <style>
        @keyframes typing {
          from { width: 0; }
          to { width: 100%; }
        }
        @keyframes blink {
          50% { border-color: transparent; }
        }
        .typewriter {
          color: #4ade80;
          font-size: 24px;
          overflow: hidden;
          white-space: nowrap;
          border-right: 3px solid #4ade80;
          animation: typing 3s steps(20), blink 0.5s step-end infinite;
        }
      </style>
      <span class="typewriter">Hello, World!</span>
    </div>
  `,
  width: 600,
  height: 100
});
```

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `url` | string | CDN URL of the generated GIF |
| `id` | string | Unique GIF identifier |
| `width` | integer | GIF width in pixels |
| `height` | integer | GIF height in pixels |
| `frameCount` | integer | Number of frames |
| `duration` | integer | Total duration in milliseconds |
| `size` | integer | File size in bytes |
| `createdAt` | string | ISO 8601 timestamp |
| `userStorageUrl` | string | URL in your storage (if configured) |

## Tips for Better GIFs

1. **Keep dimensions reasonable** - Smaller GIFs load faster and have better compression
2. **Optimize animations** - Simple, smooth animations compress better than complex ones
3. **Use solid backgrounds** - Gradients and transparency increase file size
4. **Limit colors** - GIFs are limited to 256 colors per frame
5. **Loop seamlessly** - Design animations to loop smoothly for better visual appeal

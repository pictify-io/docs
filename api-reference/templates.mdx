# Templates

Templates are reusable designs with dynamic variables. Create once, render with different data.

## List Templates {#list}

Retrieve all templates in your account.

<CodeGroup>
```bash cURL
curl https://api.pictify.io/v1/templates \
  -H "Authorization: Bearer $API_KEY"
```

```typescript Node.js
const templates = await pictify.templates.list({
  page: 1,
  limit: 20
});
```

```python Python
templates = pictify.templates.list(page=1, limit=20)
```
</CodeGroup>

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | integer | 1 | Page number |
| `limit` | integer | 20 | Results per page (max: 100) |

### Response

```json
{
  "templates": [
    {
      "uid": "tmpl_abc123",
      "name": "Social Card",
      "description": "Open Graph image template",
      "width": 1200,
      "height": 630,
      "variables": ["title", "description", "author"],
      "createdAt": "2026-01-15T10:00:00Z",
      "updatedAt": "2026-01-28T14:30:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 45,
    "totalPages": 3,
    "hasNext": true,
    "hasPrev": false
  }
}
```

## Create Template {#create}

Create a new template.

<CodeGroup>
```bash cURL
curl -X POST https://api.pictify.io/v1/templates \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Blog Post Card",
    "description": "Social share image for blog posts",
    "html": "<div style=\"width: 1200px; height: 630px; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); padding: 60px; display: flex; flex-direction: column; justify-content: center;\"><h1 style=\"color: white; font-size: 48px; margin: 0;\">{{title}}</h1><p style=\"color: rgba(255,255,255,0.8); font-size: 24px; margin-top: 20px;\">{{description}}</p><div style=\"margin-top: auto; color: white;\">By {{author}}</div></div>",
    "width": 1200,
    "height": 630
  }'
```

```typescript Node.js
const template = await pictify.templates.create({
  name: 'Blog Post Card',
  description: 'Social share image for blog posts',
  html: `
    <div style="width: 1200px; height: 630px; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); padding: 60px; display: flex; flex-direction: column; justify-content: center;">
      <h1 style="color: white; font-size: 48px; margin: 0;">{{title}}</h1>
      <p style="color: rgba(255,255,255,0.8); font-size: 24px; margin-top: 20px;">{{description}}</p>
      <div style="margin-top: auto; color: white;">By {{author}}</div>
    </div>
  `,
  width: 1200,
  height: 630
});
```

```python Python
template = pictify.templates.create(
    name='Blog Post Card',
    description='Social share image for blog posts',
    html='''
    <div style="width: 1200px; height: 630px; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); padding: 60px; display: flex; flex-direction: column; justify-content: center;">
      <h1 style="color: white; font-size: 48px; margin: 0;">{{title}}</h1>
      <p style="color: rgba(255,255,255,0.8); font-size: 24px; margin-top: 20px;">{{description}}</p>
      <div style="margin-top: auto; color: white;">By {{author}}</div>
    </div>
    ''',
    width=1200,
    height=630
)
```
</CodeGroup>

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Template name |
| `description` | string | No | Template description |
| `html` | string | * | HTML content with `{{variable}}` placeholders |
| `fabricJSData` | object | * | FabricJS canvas data |
| `width` | integer | Yes | Template width in pixels |
| `height` | integer | Yes | Template height in pixels |

<Note>
Either `html` or `fabricJSData` is required, but not both.
</Note>

### Response

```json
{
  "uid": "tmpl_xyz789",
  "name": "Blog Post Card",
  "description": "Social share image for blog posts",
  "width": 1200,
  "height": 630,
  "variables": ["title", "description", "author"],
  "createdAt": "2026-01-29T10:30:00Z",
  "updatedAt": "2026-01-29T10:30:00Z"
}
```

## Update Template {#update}

Update an existing template.

<CodeGroup>
```bash cURL
curl -X PUT https://api.pictify.io/v1/templates/tmpl_abc123 \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Social Card",
    "html": "<div>Updated HTML...</div>"
  }'
```

```typescript Node.js
const template = await pictify.templates.update('tmpl_abc123', {
  name: 'Updated Social Card',
  html: '<div>Updated HTML...</div>'
});
```

```python Python
template = pictify.templates.update('tmpl_abc123',
    name='Updated Social Card',
    html='<div>Updated HTML...</div>'
)
```
</CodeGroup>

### Request Body

All fields are optional. Only include fields you want to update.

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | string | New template name |
| `description` | string | New description |
| `html` | string | New HTML content |
| `fabricJSData` | object | New FabricJS data |
| `width` | integer | New width |
| `height` | integer | New height |

## Render Template {#render}

Render a template with variables to generate an image.

<CodeGroup>
```bash cURL
curl -X POST https://api.pictify.io/v1/templates/tmpl_abc123/render \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "variables": {
      "title": "My Blog Post",
      "description": "An interesting article about APIs",
      "author": "Jane Smith"
    },
    "format": "png"
  }'
```

```typescript Node.js
const result = await pictify.templates.render('tmpl_abc123', {
  variables: {
    title: 'My Blog Post',
    description: 'An interesting article about APIs',
    author: 'Jane Smith'
  },
  format: 'png'
});

console.log(result.url);
```

```python Python
result = pictify.templates.render('tmpl_abc123',
    variables={
        'title': 'My Blog Post',
        'description': 'An interesting article about APIs',
        'author': 'Jane Smith'
    },
    format='png'
)
```
</CodeGroup>

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `variables` | object | No | Variables to inject into template |
| `format` | string | No | Output format: `png`, `jpeg`, `webp` (default: `png`) |
| `quality` | integer | No | Quality for JPEG/WebP (1-100) |
| `deviceScaleFactor` | number | No | Device scale for retina (1-3) |

### Response

```json
{
  "url": "https://cdn.pictify.io/renders/img_abc123.png",
  "id": "img_abc123",
  "templateUid": "tmpl_abc123",
  "width": 1200,
  "height": 630,
  "format": "png",
  "createdAt": "2026-01-29T10:30:00Z"
}
```

## Get Template Variables {#variables}

Get the list of variables defined in a template.

<CodeGroup>
```bash cURL
curl https://api.pictify.io/v1/templates/tmpl_abc123/variables \
  -H "Authorization: Bearer $API_KEY"
```

```typescript Node.js
const variables = await pictify.templates.getVariables('tmpl_abc123');
```

```python Python
variables = pictify.templates.get_variables('tmpl_abc123')
```
</CodeGroup>

### Response

```json
{
  "variables": [
    {
      "name": "title",
      "type": "string",
      "required": true
    },
    {
      "name": "description",
      "type": "string",
      "required": false
    },
    {
      "name": "author",
      "type": "string",
      "required": false
    },
    {
      "name": "showBadge",
      "type": "boolean",
      "required": false
    }
  ]
}
```

## Variable Types

Templates support different variable types:

| Type | Example | Description |
|------|---------|-------------|
| `string` | `"Hello World"` | Text content |
| `number` | `42`, `3.14` | Numeric values |
| `boolean` | `true`, `false` | Conditional rendering |
| `array` | `["a", "b"]` | Lists for iteration |
| `object` | `{name: "..."}` | Nested data |

## Using Variables in Templates

### Simple Interpolation

```html
<h1>{{title}}</h1>
<p>By {{author}}</p>
```

### Conditional Rendering

```html
<div _if="showBadge" class="badge">Premium</div>
```

### Expressions

```html
<p>Total: {{currency(price * quantity, 'USD')}}</p>
```

See [Expressions](/concepts/expressions) for the full expression syntax.
